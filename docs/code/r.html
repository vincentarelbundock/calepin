R

Calepin embeds R via extendr, running code directly in process. Variables, loaded packages, and state persist across chunks.

Basic usage

fit <- lm(mpg ~ wt + hp, data = mtcars)
summary(fit)$coefficients
>                Estimate Std. Error   t value     Pr(>|t|)
> (Intercept) 37.22727012 1.59878754 23.284689 2.565459e-20
> wt          -3.87783074 0.63273349 -6.128695 1.119647e-06
> hp          -0.03177295 0.00902971 -3.518712 1.451229e-03

Inline expressions

The median horsepower is 123 and the dataset has 32 observations. Numeric results are automatically formatted with 3 significant digits and comma separators.

Plots

R plots are captured automatically. Use chunk options to control dimensions and format.

plot(mtcars$wt, mtcars$mpg,
     xlab = "Weight", ylab = "MPG",
     main = "Fuel efficiency by weight",
     pch = 19, col = "steelblue")

Warnings and messages

By default, warnings and messages are displayed. Suppress them with chunk options:

log(-1)
> [1] NaN
