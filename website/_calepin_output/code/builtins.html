Built-in Functions

Calepin provides built-in functions using the bracketed span syntax: <span class="class">content</span>. These produce format-appropriate output for HTML, LaTeX, Typst, and Markdown.

pagebreak

<div style="page-break-after: always;"></div> inserts a page break (<div style="page-break-after: always;"> in HTML, \newpage{} in LaTeX, #pagebreak() in Typst).

video

Embed a video or streaming service. YouTube and Vimeo URLs are automatically converted to embed URLs.

[]{.video url="https://www.youtube.com/watch?v=dQw4w9WgXcQ"}
[]{.video url="https://vimeo.com/123456" width="640" height="480"}
Attribute Default Description
url required Video URL (YouTube, Vimeo, or direct)
width "100%" Player width
height "400" Player height
title "Video" Accessible title

lorem

Insert placeholder lorem ipsum text. Default: 1 paragraph.

[]{.lorem}
[]{.lorem paragraphs=3}
[]{.lorem sentences=5}
[]{.lorem words=20}

Example with 2 sentences: Lorem ipsum dolor sit amet consectetur adipiscing elit. Elit sed do eiusmod tempor incididunt ut labore et dolore magna.

placeholder

Insert a placeholder image for layout prototyping. Produces an inline SVG in HTML, \fbox in LaTeX, #rect in Typst.

[]{.placeholder}
[]{.placeholder width=300 height=200}
[]{.placeholder width=400 height=100 text="Banner" color="#e0e0ff"}

Attribute Default Description
width 600 Width in pixels
height 400 Height in pixels
text "WxH" Display text
color "#cccccc" Background color

Custom functions

Create your own bracketed span functions by adding a template in _calepin/partials/{engine}/. For example, to create a <span class="highlight">text</span> function, add _calepin/partials/html/highlight.html:

<mark class="highlight">{{ content }}</mark>

Then use it in your document: <span class="highlight">important text</span>.

The template receives all span attributes as variables, plus content, class, classes, id, and format.
