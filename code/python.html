Python

Calepin embeds Python via PyO3, running code directly in process. A shared globals dictionary persists variables across chunks.

Basic usage

```{python, chunk-1}
#| echo = fenced
import math
primes = [2, 3, 5, 7, 11, 13]
print(f"Sum: {sum(primes)}, Mean: {sum(primes)/len(primes):.1f}")
```
> Sum: 41, Mean: 6.8

Inline expressions

The square root of 2 is 1.4142.

Persistent state

Variables defined in one chunk are available in later chunks:

message = "Hello from Python"
print(message.upper())
> HELLO FROM PYTHON

Plots

Matplotlib figures are captured automatically:

import matplotlib.pyplot as plt
import math

x = [i / 10 for i in range(63)]
y = [math.sin(v) for v in x]
plt.plot(x, y)
> []
plt.title("Sine wave")
> Text(0.5, 1.0, 'Sine wave')
plt.xlabel("x")
> Text(0.5, 0, 'x')
plt.ylabel("sin(x)")
> Text(0, 0.5, 'sin(x)')
plt.show()
